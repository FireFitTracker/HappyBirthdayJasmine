# Jasmine's Birthday Website

A simple, interactive static site built as a birthday gift for Jasmine.

## Files

- `index.html`: page structure and written message
- `styles.css`: visual design and responsive layout
- `script.js`: interactive moments (letter reveal, rotating messages, wish sky, confetti)

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. In this folder, run:

```bash
git init
git add -A
git commit -m "Create Jasmine birthday website"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

3. In GitHub, enable Pages from the `main` branch root.

## Personalize It

Edit the text in `index.html` and the rotating reasons in `script.js` to make it even more specific to Jasmine.
